# 🤖 TaskFlow AI

A modern AI-powered Task Management Landing Page built using HTML, CSS, and JavaScript.

The website features a premium UI with responsive design, glassmorphism effects, dark/light mode, animations, and interactive components.

## ✨ Features

- 🚀 Responsive Design
- 🌙 Dark / Light Mode
- 🎯 Sticky Navigation Bar
- 📊 Animated Statistics Counter
- 💫 Scroll Animations
- 🔝 Scroll to Top Button
- ⌨️ Typing Effect
- 🎨 Glassmorphism UI
- 🌈 Gradient Buttons
- 📱 Mobile Friendly Layout
- 📬 Contact Form
- 🔗 Social Media Links
